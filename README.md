# 鲍叔律师团队 · 律所一体化知识库（BaoShu Law OS）· v1.0

本仓库用于沉淀和管理鲍叔律师团队在“工厂货款追讨”领域的完整方法论与业务资料，
构建一套可复制、可培训、可扩展的律所一体化作业系统。

## 一、整体结构总览（一级目录）

1. `1_流量与获客`  
   - 客户来源、活动设计、朋友圈内容、视频号策略等。

2. `2_邀约体系`  
   - 从“加微信 → 进入私域 → 约见/约电话”的标准话术与流程。

3. `3_成交体系`  
   - 篮子报价、风险评估、签约话术、收费模式设计等。

4. `4_办案体系`  
   - 非诉、诉讼、执行、刑事四大模块的完整 SOP。

5. `5_案管体系`  
   - 案件台账、进度跟踪、节点提醒、客户反馈机制。

6. `6_运营体系`  
   - 团队分工、绩效规则、品牌运营、AI/知识库维护。

7. `7_产品体系`  
   - 书+课+陪跑+咨询+诉讼+执行+刑事的产品化设计。

8. `8_立体网查系统`  
   - 线上立体网查 SOP、评估表模板、风险分级模型。

9. `9_法律文书模板`  
   - 起诉状、答辩状、法律意见书、和解协议、备忘录等。

10. `10_AI工具与SOP`  
    - ChatGPT 使用规范、常用提示词模板、自动化流程设计。

11. `11_内容与视频生产体系`  
    - 朋友圈文案模板、视频脚本模板、案例故事结构。

12. `12_团队制度与培训`  
    - 新人培训手册、办案规范、服务标准。

13. `13_客户沟通手册`  
    - 首次咨询、维权策略说明、费用解释、风险提示话术。

14. `14_执行与和解体系`  
    - 执行阶段策略、和解谈判模板、分期还款协议模型。

15. `15_课程与活动体系`  
    - 工厂老板课、社群活动、线上直播、诊断/评估活动设计。

---

## 二、使用原则

1. 所有文件尽量使用 **Markdown 格式（.md）**，便于阅读与版本管理。  
2. 每个一级目录下，至少包含一个 `README.md` 用于说明该部分的结构与目的。  
3. 涉及具体客户、案件信息时，统一做脱敏处理，避免隐私与合规风险。  
4. 本知识库主要记录「方法论 + 模板 + SOP」，不做案件档案库使用。
